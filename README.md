# create-vanilla-app

## Vanilla JS App starter pack with modern tools
```
- npx degit thenitesnarang/create-vanilla-app yourAppName
- cd yourAppName
- npm intall
- npm run start
```

## features
- Import/Export
- Minification
- Hashing
- Babel Support
- Inline image support
- CSS Loader
- Code Splitting
- Route Based Code Splitting
